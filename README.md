# TeamProjectTemporary
Временный репозиторий для работы над проектом "Разработка сервиса проверки стойкости паролей и их присутствия в базах утечек"

## Общее описание
Проект представляет собой веб‑приложение для проверки стойкости пароля.  
Оценка выполняется по следующим критериям:
- длина пароля;
- состав (цифры, буквы в разных регистрах, спецсимволы);
- отсутствие запрещённых последовательностей (простые/клавиатурные/популярные пароли, повторы, даты);
- отсутствие пароля в базе утекших хешей.

Результат проверки отображается пользователю в виде **«Слабый» / «Средний» / «Сильный»** с пояснением причин.

## MVP (Minimum Viable Product)

Минимально жизнеспособный продукт включает:

1. **Веб-интерфейс** — одна страница с полем ввода пароля и кнопкой проверки
2. **API проверки стойкости** — анализ пароля по критериям:
   - длина (минимум 8 символов)
   - наличие цифр, строчных и заглавных букв, спецсимволов
   - отсутствие простых/клавиатурных/популярных последовательностей
   - отсутствие повторов и дат
3. **База утекших хешей** — проверка пароля по SQLite-базе (только хеши, не пароли)
4. **Вывод результата** — оценка «Слабый» / «Средний» / «Сильный» с пояснением причин
