# Инструкция по запуску проекта перед предзащитой

## Цель
Этот файл нужен для быстрого запуска проекта перед предзащитой, чтобы сервис можно было открыть локально и при необходимости показать с других устройств через публичную ссылку.

## 2. Запустить backend и frontend
Запустить сервер FastAPI:

```powershell
python -m uvicorn backend.main:app --host 127.0.0.1 --port 8000
```

После запуска локальная версия будет доступна по адресу:

```text
http://127.0.0.1:8000
```

## 3. Проверить локальный запуск
Открыть в браузере:

```text
http://127.0.0.1:8000
```
## 4. Запустить внешнюю ссылку для демонстрации с других устройств
Если нужно открыть проект с другого устройства или из другой сети, запустить туннель:

```powershell
npx.cmd localtunnel --port 8000
```

После запуска в консоли появится строка вида:

```text
your url is: https://example-name.loca.lt
```

Именно эту ссылку нужно открыть на другом устройстве.


## 5. Что важно не закрывать
Во время демонстрации должны оставаться запущенными:
- окно с `uvicorn`;
- окно с `localtunnel`.

Если закрыть любое из этих окон, сайт по публичной ссылке перестанет открываться.

## 6. Рекомендуемый порядок запуска перед показом
1. Открыть терминал в папке проекта.
2. Запустить `uvicorn`.
3. Проверить локально, что сайт работает.
4. В новом окне терминала запустить `npx.cmd localtunnel --port 8000`.
5. Скопировать публичную ссылку.
6. Проверить ссылку на телефоне или другом устройстве.
7. Только после этого начинать демонстрацию.
