# РУЧНОЕ ТЕСТИРОВАНИЕ

## Инструкция
1. Открыть сайт
2. Ввести пароль из таблицы
3. Нажать кнопку "Проверить"
4. Записать результат в колонку "Фактический результат"
5. Если результат не совпадает с ожидаемым — отметить баг
