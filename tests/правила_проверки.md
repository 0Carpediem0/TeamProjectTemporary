# ПРАВИЛА ПРОВЕРКИ ПАРОЛЯ

## 1. ДЛИНА

| Длина | Баллы |
|-------|-------|
| Меньше 8 символов | СЛАБЫЙ (автоматически) |
| 8-11 символов | +1 балл |
| 12+ символов | +2 балла |

> Баллы = числовой рейтинг сложности пароля. Чем больше баллов, тем сложнее пароль.

---

## 2. СОСТАВ

| Правило | Баллы |
|---------|-------|
| Только цифры | -1 (штраф) |
| Только буквы в нижнем регистре | -1 (штраф) |
| Только буквы (нет цифр) | -1 (штраф) |
| Есть цифры | +1 |
| Есть заглавные буквы | +1 |
| Есть спецсимволы (! @ # $ % ^) | +1 |

---

## 3. ЗАПРЕЩЕННЫЕ ПОСЛЕДОВАТЕЛЬНОСТИ

| Что проверяем | Примеры | Штраф |
|--------------|---------|-------|
| Простая последовательность | 123, 1234, 12345 | -1 |
| Клавиатурная | qwerty | -1 |
| Популярные пароли | password, admin | -1 |
| Повторы | 111, aaaa | -1 |
| Годы, даты | 1990, 2020 | -1 |

> **Примечание:** если пароль содержит 2 и более запрещенных последовательности → штраф -2 (или автоматический СЛАБЫЙ)

---

## 4. ПРОВЕРКА ПО БАЗЕ УТЕЧЕК

| Результат | Действие |
|-----------|----------|
| Пароль найден в базе хэшей утечек | СЛАБЫЙ (автоматически) |

---

## 5. ИТОГОВАЯ ОЦЕНКА

**Сначала проверяем автоматический СЛАБЫЙ:**
- Длина < 8 символов → СЛАБЫЙ
- Пароль в базе утечек → СЛАБЫЙ

**Если автоматический СЛАБЫЙ не сработал, считаем баллы:**

| Сумма баллов | Оценка |
|--------------|--------|
| 0-1 | СЛАБЫЙ |
| 2-3 | СРЕДНИЙ |
| 4+ | СИЛЬНЫЙ |

---
