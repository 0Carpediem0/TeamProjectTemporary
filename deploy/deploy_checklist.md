# Чек-лист деплоя

## Перед загрузкой проекта

- приложение запускается локально
- `requirements.txt` актуален
- папки `backend` и `frontend` присутствуют
- база утечек находится в `backend/data`
- в проект не включены секреты и конфиденциальные данные

## На сервере

- есть доступ по SSH
- установлен `python3`
- установлен `pip`
- можно создать виртуальное окружение
- подготовлена директория для проекта

## После установки

- зависимости установились без ошибок
- `uvicorn` запускается
- главная страница открывается
- `POST /api/check` отвечает корректно
- статика загружается без ошибок
- лог приложения создаётся и читается

## Перед демонстрацией

- известен адрес приложения
- проверены 2-3 тестовых пароля
- есть способ быстро перезапустить сервис
- есть доступ к логам на случай диагностики
